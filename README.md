# otto-finance — macOS App Releases

Real-time trading signal dashboard for stocks, MES futures, and Binance crypto.
Connects to a local Python WebSocket server and displays live signals with confidence gauges.

---

## How to install

1. Download the zip for the latest version below
2. Unzip — drag **otto_finance_app.app** to your **Applications** folder
3. First launch: right-click the app → **Open** → **Open** (bypasses Gatekeeper on unsigned apps)
4. Enter your server URL in Settings (gear icon ⚙)

> macOS may say the app is from an unidentified developer. Right-clicking and choosing Open bypasses this once.
